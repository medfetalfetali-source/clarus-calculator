Pré-triagem assistencial

Versão completa com lógica de escalonamento composto. Exames como Doppler, Perfil Biofísico e Crescimento permanecem base A, mas sobem rapidamente quando aparecem sinais contextuais de maior carga cognitiva.

Calculadora rápida de idade gestacional

Uso sugerido: a secretária pode estimar a IG de hoje pela DUM ou pelo ultrassom do 1º trimestre. A ferramenta também identifica janelas típicas de exames para facilitar o reconhecimento da fase gestacional.
Preencha a DUM e/ou os dados do ultrassom do 1º trimestre para estimar a idade gestacional de hoje, a DPP e a janela típica do exame.
Janelas cadastradas:
  • 11–13s6d → Morfológico 1º trimestre / Translucência
  • 20–24s → Morfológico 2º trimestre
  • 24–28s → Ecocardiograma fetal
  • 32–36s → Morfológico 3º trimestre
  • 37–39s → Vitalidade fetal + Doppler + PBF

1. Situação atual

Primeiro perguntamos sobre sinais atuais concretos. Isso evita alarmismo, melhora a precisão da triagem e protege a paciente de uma comunicação que aumente ansiedade sem acrescentar clareza.

3. Refinadores de complexidade

Este bloco só aparece quando há chance real de maior carga cognitiva. A lógica é simples: não sobrecarregar a triagem de rotina, mas abrir refinadores quando o caso pode exigir mais tempo, unidade específica ou especialista.
